ES6中引入了class类创建对象的方法。之前学C++的时候觉得面向对象还是蛮容易理解的，在刚开始接
触了ES 的OOP好像总是有什么东西没弄明白，使用起来总会被卡住。今天看了ES6的class语法之后有种
茅塞顿开的感觉。

基本语法

  • constructor 构造函数

通过new命令创建实例的时候自动调用这个构造方法。如果不定义默认是空的构造函数。
默认返回值是：实例对象this。

创建类的方法

  • 类声明方法
  • 表达式方法

Class继承

class类模块之间可以通过extends 关键字来实现继承。

  • 基本用法

  • super关键字

在子类的构造函数中必须调用父类的super 构造函数。这是ES6实现继承的机制。

  1. 创建父类实例对象 获得父类this实例，这里super=this
  2. 子类在父类的this实例上面添加修改属性和方法
  • this关键字

子类中的this是由父类构造函数创建来的

class的静态方法 于静态属性

静态方法和静态属性指的是类的方法和属性，只能通过类名+方法/属性 来访问。

  • 基本用法

声明静态方法就是用static关键字声明方法.静态属性则不能写在class 声明内部，内部只能写方法。

其它

  1. 子类的属性
  • prototype的proto属性

指向父类的prototype属性。表示方法的继承
colorPoint.prototype.proto =Point.prototype

  • proto 属性

指向父类。表示构造函数的继承。
colorPoint.proto = Point
