GitHub Fork功能使用总结

Feb 1, 2017 | 工具 | 阅读

在博主还是菜鸟的笨鸟的时候GitHub只是作为代码托管工具用，并没有去了解它其它的功能。后来自己慢慢的去琢磨它提供的各种其它的功能，惊喜不断!今天要说的是GitHub Fork的使用。

GitHub Fork 简介

在GitHub上fork这个功能用来为开源项目来贡献代码以及同步原作者的修改的。

使用

  • fork一个项目

这个项目会出现在个人的github主页上面，其实这个功能就是从别人的项目复制一份到你的github上面

  • 提交你的修改到个人副本项目

fork之后你在你的github上面把这个项目clone到你的本地进行修改，修改之后提交到你的github相对的项目中

  • 提交自己的修改给原项目作者

如果觉得有必要可以提交自己的修改给原作者。fork过来的项目都有两个特殊的按钮
