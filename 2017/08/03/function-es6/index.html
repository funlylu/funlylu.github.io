为参数设定默认值

这个扩展还是比较实用的，与其它传统语言倒是接近了那么一小步。

注：看了上面哪个报错的例子，我们可以看出【已经设定了默认值且后面还有其它参数】在函数调用时这个参数是不能省的。所以一般我们为函数的参数设定默认值的时候应该遵循【靠右原则】，即要设定默认值的参数靠后写，不然这个参数的默认值永远无法生效，因为每次调用要给它传参，不然就报错。在C里面如果不遵循这个规则在函数声明的时候就已经报错。

函数的length属性

函数的length属性的函数是，函数期待传入的参数的个数。如果为某个参数指定了默认值，那么length将不包括这个参数。
